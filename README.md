EchoMessenger 
 
주요기능 
1. 텍스트 입력 및 출력 
2. 텍스트 입력 시간 표기 
3. 입력된 텍스트의 총계 표기 
4. 자동 텍스트 가공 
5. 자동 포커스 
6. 공백 입력 제한
7. 메세지 삭제 기능
8. 대화기록 삭제 시능
9. 글자 수 제한
